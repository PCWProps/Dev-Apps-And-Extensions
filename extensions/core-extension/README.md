## Features

- **Hello World Command**: Say hello to your friendly bear!
- **Info Display**: Learn more about BearBuddy
- **Telemetry**: Anonymous usage statistics to improve the extension (can be disabled)

## Getting Started

1. Install the extension from the VS Code Marketplace
2. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) to open the command palette
3. Type "BearBuddy" to see available commands

## Commands

- `BearBuddy: Hello World` - Display a friendly greeting
- `BearBuddy: Show Info` - Display extension information

## Configuration

This extension contributes the following settings:

- `bearbuddy.enableTelemetry`: Enable/disable anonymous telemetry (default: `true`)
- `bearbuddy.showWelcomeMessage`: Show welcome message on activation (default: `true`)

## Getting Started

1. Install the extension from the VS Code Marketplace
2. Enter your license key: Press `Ctrl+Shift+P` and type "BearBuddy Pro: Enter License Key"
3. Start using pro features!

## Commands

- `BearBuddy Pro: Hello World` - Display a friendly greeting
- `BearBuddy Pro: Enter License Key` - Enter your license key
- `BearBuddy Pro: Check License Status` - Check your current license
- `BearBuddy Pro: Advanced Feature` - Use advanced pro features (requires license)

## Configuration

This extension contributes the following settings:

- `bearbuddy-pro.licenseKey`: Your BearBuddy Pro license key
- `bearbuddy-pro.enableTelemetry`: Enable/disable anonymous telemetry (default: `true`)

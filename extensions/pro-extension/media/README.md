# BearBuddy Pro Extension Media

This directory contains icons and media assets for the BearBuddy Pro extension.

## Required Icons

- `icon.png` - 128x128px extension icon (Pro version with distinct styling)
- `icon@2x.png` - 256x256px high-resolution icon
- `banner.png` - 1280x640px marketplace banner
- `badge-pro.png` - Pro badge overlay

## Branding Differentiation

Pro extension icons should:
- Include "Pro" badge or indicator
- Use premium color scheme (gold, platinum accents)
- Maintain consistency with core extension
- Signal premium features visually

## Usage

Icons should be referenced in `package.json`:

```json
{
  "icon": "media/icon.png"
}
```

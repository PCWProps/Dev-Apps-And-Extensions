## Installation

### From VSIX (Development)
1. Build the extension: `npm run build`
2. Package the extension: `npm run package`
3. Install the generated `.vsix` file in VSCode

### From Marketplace (Coming Soon)
Search for "BearBuddy Pro" in the VSCode Extensions marketplace.

## Usage

1. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
2. Type "BearBuddy Pro" to see available commands
3. Select a command to execute

## Available Commands

- `BearBuddy Pro: Hello World` - Display a friendly greeting
- `BearBuddy Pro: Advanced Feature` - Access pro-exclusive functionality

## Installation

### 1. Purchase License

Visit [bearbuddy.dev/pricing](../pricing.html) to select and purchase your license.

### 2. Install Extension

From VS Code:
1. Open Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for "BearBuddy Pro"
3. Click "Install"

Or install from [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=bearbuddy.pro-extension).

### 3. Activate License

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Run "BearBuddy: Enter License Key"
3. Paste your license key
4. Click "Activate"

Your license will be validated and Pro features will be enabled immediately.

## Usage

### Accessing Pro Features

Once activated, Pro features are available through:

- Command Palette (search "BearBuddy Pro")
- Right-click context menus
- Status bar items
- Settings panel

### Premium Commands

- `bearbuddy.premiumFeature` - Access premium features
- `bearbuddy.codeAnalysis` - Run advanced code analysis
- `bearbuddy.teamSync` - Sync with team resources

### Configuration

Pro-specific settings:

```json
{
  "bearbuddy.pro.enableAdvancedAnalysis": true,
  "bearbuddy.pro.teamSync": true,
  "bearbuddy.pro.theme": "premium-dark"
}
```
